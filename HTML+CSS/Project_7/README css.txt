3d Box📦    # 🧊 3D Cube Project

This project demonstrates a **3D Cube** built using **HTML & CSS**.  
Each face of the cube is labeled and styled with 3D transformations.

---

## ✨ Features
- 🎲 Six cube faces: Front, Back, Right, Left, Top, Bottom  
- 🎨 Styled with CSS for 3D effect  
- 📱 Responsive cube container  
- 🌍 Can be rotated using CSS transforms
